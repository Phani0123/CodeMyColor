Frequently Asked Questions

How do I copy a color code?

+

To copy a color code, simply click on the copy icon next to the code you want to use. You'll see a notification confirming that the code has been copied to your clipboard. You can then paste it into your project.

What's the difference between RGB and RGBA?

+

RGB (Red, Green, Blue) defines colors using three values that represent the intensity of red, green, and blue components. RGBA adds a fourth value for alpha (transparency), allowing you to create semi-transparent colors. The alpha value ranges from 0 (completely transparent) to 1 (completely opaque).

How do I create a gradient?

+

To create a gradient, visit our Gradient Generator tool. You can select two or more colors and adjust their positions. The tool will generate the CSS code for your gradient, which you can then copy and use in your project. You can create linear gradients (in any direction) or radial gradients.

Can I save my favorite colors?

+

Yes! You can save your favorite colors by clicking the "Add to Favorites" button when you've created a color you like. Your favorites are stored in your browser's local storage, so they'll be available the next time you visit. You can access your saved colors from the "Favorites" section on each tool page.

Which browsers are supported?

+

Our tools work on all modern browsers, including Chrome, Firefox, Safari, and Edge. We recommend keeping your browser updated to the latest version for the best experience. Some features may not work correctly on older browsers like Internet Explorer.

How do I convert between color formats?

+

When you create a color using any of our tools, we automatically generate the code in multiple formats (RGB, HEX, HSL, etc.). You can simply copy the format you need. If you want to convert an existing color, enter it in any of our color generators, and we'll show you all the equivalent codes in other formats.

Are the color tools free to use?

+

Yes, all our color tools are completely free to use. You can generate as many colors as you need, save your favorites, and use them in your projects without any limitations.

How do I report a bug?

+

If you encounter any issues while using our tools, please use the contact form below to report the bug. Include as much detail as possible, such as the browser you're using, the specific tool where you found the issue, and steps to reproduce the problem. Screenshots are also helpful.
