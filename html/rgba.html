How to Use RGBA Colors in CSS

Text Color with Transparency

color: rgba(255, 0, 0, 0.8);

This text is colored using RGBA with transparency.

Background Color with Transparency

background-color: rgba(255, 0, 0, 0.5);
This element has an RGBA background color with 50% opacity.

Box Shadow with Transparency

box-shadow: 0 5px 15px rgba(255, 0, 0, 0.5);
This element has a semi-transparent red shadow.

Border with Transparency

border: 3px solid rgba(255, 0, 0, 0.8);
This element has a semi-transparent red border.

Text Shadow with Transparency

text-shadow: 2px 2px 4px rgba(255, 0, 0, 0.7);

This text has a semi-transparent red shadow.

Gradient with Transparency

background: linear-gradient(
  to right,
  rgba(255, 0, 0, 0.8),
  rgba(0, 0, 255, 0.5)
);
This element has a gradient background with transparency.
